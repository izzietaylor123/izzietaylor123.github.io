    Projects

            
            
              

    Tenfins Portal

                 

                  This project was a culmination of my entire degree at CU Boulder.               This project was worked on with a significant team, where I took on a team lead position.               In this project we worked to create a hub for flutter developers to come together and share projects, create, and collaborate.               This project had very loose leadership, which was a challenge, but also allowed me to grow as someone leading a team but also take the project by the reigns and learn the things I wanted to.               The scope of the project was a messaging and channels system built on a GoLang backend, talking to a postgres database.               Some of the other goals for this project were to make some thing scalable and easy to maintain.               From this we were able to create a product built on GCP using Kubernetes to scale the application and containers, that are deployed to maintain other AI services we built as well.               Not only was I able to lead a team distributing work and responsibilities, but I was able to learn a lot for myself, from tools like Typescript to Kubernetes.               This was a great project to work on, and I am proud of the work we were able to accomplish despite the challenges we faced!             

              
              

    NCAA to NBA player analysis

                 

                  This project was a final project for my Data Mining class at CU Boulder.               This project was worked on with a couple of other friends, and was a great learning experience overall.               In this project we used a multitude of data science techniques to analyze the relationship between NCAA and NBA players to see if there was a correlation between performance in college and the NBA as well as determining a success statistic for this relationship.               Though this project was split up into a few parts, one of the hardest parts was the collection and cleaning of the data. The data was not very clean, and had to be cleaned and processed to get it into a usable format.               Our results gave us a solid statistic to determine success, and did notice some correlation between areas and types of players that were successful there.               We did have some disconnect in our ML models, which only got to about 85% accuracy, but overall I think we did a great job with the time we had!             

              
                

    K-Means Project

                 

                  This project was a final project for my Advnced Data Science class at CU Boulder.               This project was worked on with my friend Kevin, and was a great learning experience overall.               In this project we implemented K-means clustering on the MNIST and Fashion-MNIST datasets and then changed parameters trying to find the best way to intialize centroids, iterate and update centroids.               This project was interesting because it was a bit difficult to determine what was actually happening.               To really understand the processes, it was important to understand not only the numerical data, but what K-means was doing which incorporates an unusual abstract complexity.               Overall, I'm really happy with how this project turned out, I learned a lot and had a lot of fun working on it!
